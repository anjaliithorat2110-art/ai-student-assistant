
Smart AI Assistant for Students 

Project Description
This project is an AI-based assistant that helps students with:
- Answering questions (Chatbot)
- Generating summaries from PDF notes

It uses Generative AI to provide intelligent and human-like responses.

---

 Features
- AI Chatbot for answering questions
- PDF Notes Summarizer
- Easy-to-use interface
- Fast and accurate responses

---

 Technologies Used
- Python
- Streamlit
- OpenAI API
- PyPDF2

---

How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

---

 Output
- Chatbot answering questions
- PDF summary generation

---
 Conclusion
This project helps students save time and improve learning using AI technology.
